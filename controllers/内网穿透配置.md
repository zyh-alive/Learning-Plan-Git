1.什么是内网穿透？飞书为什么不需要？支付宝为什么需要？
-->将本地ip转化成一个外部服务器能够访问到的地址并有访问通道
-->飞书定时播报只需要本地服务器向飞书发送消息，飞书不需要向本地发送，所以不需要公网IP
-->支付宝需要向本地回调，因此需要一个本地的地址

2.配置
-->mac/cursor终端输入 brew install cloudflared
-->查看版本 cloudflared --version
-->查看状态 ps aux | grep cloudflared
alg              49091   0.0  0.2 412069360  36352 s006  S+    9:27上午   0:10.16 cloudflared tunnel --url http://localhost:3003
类似于这条输出为正常
-->永久配置公网ip 
# 1. cloudflared tunnel login 只是授权你的账号，不是「永久公网 IP」
# 2. 临时隧道（quick tunnel）每次重启域名都会变
# 3. 想要固定域名，需要：
#    a. 有自己的域名 + 接入 Cloudflare DNS
#    b. 创建命名隧道：cloudflared tunnel create my-tunnel
#    c. 绑定域名：cloudflared tunnel route dns my-tunnel demo.你的域名.com
-->临时使用  cloudflared tunnel --url http://localhost:3003（本地地址），可能不稳定，且重启时需要更改环境变量（每次产生的地址不同）
-->针对连接不稳定，使用http2   cloudflared tunnel --url http://localhost:3003 --protocol http2

3.路由配置常见错误
# - 路径写错：/api/payment/alipay/callback（代码监听的是 /api/pay/notify/alipay）
# - 域名没更新：重启 tunnel 后域名变了，但配置没改
# - 有空格：https://xxx /api/...（= 后面不能有空格）
